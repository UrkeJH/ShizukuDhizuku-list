# 説明
- ShizukuまたはDhizukuをサポートするオープンソースソフトウェアのみを収集しています
- 一部の中国語説明は英語の機械翻訳です

# Shizukuについて
Shizukuは、一般のアプリがapp_processによって起動されたJavaプロセスを利用して、adbまたはroot権限でシステムAPIを直接利用できるようにするツールです。

[Shizuku有効化ガイド](https://shizuku.rikka.app/ja/guide/setup/)

# Dhizukuについて
Shizukuの設計思想を参考にして、DeviceOwner（デバイスオーナー）権限を他のアプリに共有します。

[Dhizuku有効化ガイド](https://github.com/iamr0s/Dhizuku/discussions/16)
