## Shizuku介紹
Shizuku 可以幫助普通應用借助一個由 app_process 啓動的 Java 進程直接以 adb 或 root 特權使用系統 API

[Shizuku激活教程](https://shizuku.rikka.app/zh-hans/guide/setup/)
[官方網站](https://shizuku.rikka.app/)

## Dhizuku介紹
參考Shizuku的設計思想，分享 DeviceOwner (設備所有者) 權限給其餘應用

[Dhizuku激活教程](https://github.com/iamr0s/Dhizuku/discussions/16)
