# Shizuku 介紹
Shizuku 可以幫助普通應用藉由一個由 app_process 啟動的 Java 行程直接以 adb 或 root 權限使用系統 API

[Shizuku 啟用教學](https://shizuku.rikka.app/zh-hant/guide/setup/)

# Dhizuku 介紹
參考 Shizuku 的設計理念，分享 DeviceOwner（裝置擁有者）權限給其他應用

[Dhizuku 啟用教學](https://github.com/iamr0s/Dhizuku/discussions/16)
